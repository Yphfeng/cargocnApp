## ✨ 主要功能

- 🌈 司机找货、物流运输、车辆加油
- 📦 货主找车、发布货源、接单提醒
- 🛡 油气交易、车辆服务、卡友互助
- ⚙️ 短导运输、运力采购、路况天气

## ✨ 使用技术
- unni-app+vuex+vue-router
- [高德定位和地图](https://lbs.amap.com/api/javascript-api-v2/summary)
- 一键登录
- 微信分享
- 二维码扫码
- 微信支付
- 消息推送
- 微信H5跳转App
